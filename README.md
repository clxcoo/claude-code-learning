# Claude Code 与 OpenClaw 学习指南

零基础入门 Claude Code 和 OpenClaw 的互动学习技能。

## 简介

本技能基于 KimYx0207/Claude-Code-x-OpenClaw-Guide-Zh 教程，帮助零基础用户系统学习 AI 编程工具。

## 特性

- 🤖 **双线学习路径**：Claude Code 编程线 + OpenClaw 助手线
- ⏱️ **快速上手**：3小时上手 Claude Code，1小时跑通 OpenClaw
- 🛡️ **避坑指南**：国内用户常见问题与解决方案
- 📚 **实际案例**：可复制操作的教学示例

## 支持的平台

### Claude Code
- Windows 10+ / macOS 10.15+ / Linux
- 支持 MiniMax、硅基流动、OpenRouter 等中转站

### OpenClaw
- Node.js 22.12.0+
- 支持 29+ AI 模型（OpenAI、Claude、Gemini、Ollama）
- 支持 15+ 消息平台（Telegram、Discord、飞书等）
